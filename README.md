# VerifyAI 🛡️
**AI-Powered Misinformation & Bias Detector**

## 1. Problem Statement
Fake news spreads rapidly on social media, influencing public opinion and creating panic. Many users cannot identify whether information is credible or manipulated.

## 2. Our Solution
VerifyAI is an intelligence hub that uses **Google Gemini** to provide:
- **Credibility Scoring:** 0-100% trust rating based on cross-referencing.
- **Bias Detection:** Identifying political and emotional leanings.
- **Multimodal Analysis:** Analyzing images and screenshots for context (Vision).
- **Fact-Check Explanations:** Clear, AI-generated reasoning for every result.

## 3. Tech Stack
- **Frontend:** Next.js (React), Tailwind CSS, Lucide Icons
- **AI Engine:** Google Gemini API (Pro & Vision)
- **Deployment:** Vercel
